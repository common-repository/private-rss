== Description ==

By Private RSS wordpress plugin you could alter your RSS feed to a public and a private feed. Summaries goes to the public feed and full posts goes to the private feed. By this you could ask money for the private RSS feed.

* In some form - like in a post - you should let your visitors know they could subscribe to an RSS feed with the full posts. Normal RSS fedd will contain only the summaries.
* You should create a link / button by what subscribers get to PayPal and they could send the money for you.
* PayPal will send you an e-mail after you have received the money.
* At the administration page of the plugin you could add the subscriberâ€™s datas.
* The subscriber will automatically gets an email with an unique URL for the private RSS feed.

Available languages: english, hungarian, russian


== Installation ==

1. Download, extract and uppload to the wp-content/plugins folder
2. Switch on the plugin
3. You will get a submenu on the plugin page
4. The plugin do the same as Feedburner Feedsmith plugin


== Frequently Asked Questions ==
= May I use this plugin with Feedburner? =
Yes, on the setting page you could define your feedburner url for the public feed and for the comments feed. For private feed you can not use feedburner stats, as they have different unique URLs.
You should disable the Feedburner Feedsmith plugin, Private RSS will do the same for you.
